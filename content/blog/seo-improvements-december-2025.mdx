# SEO & Discoverability Improvements

We've made significant improvements to help you find our content more easily and share it with others.

## What's New

### Enhanced Social Sharing

- **LinkedIn & Twitter Cards** - Articles now display rich previews when shared
- **Social Share Buttons** - Easily share content on Twitter, LinkedIn, Reddit, and Hacker News
- **Floating Share Bar** - Mobile-friendly sharing from any article

### FAQ Rich Snippets

Added FAQ schema markup to help our content appear in Google's rich results:

- Homepage FAQs about Claude Code basics
- Track-specific FAQs for each learning path
- Better visibility in search results

### AI Discoverability

- **llms.txt** - Added machine-readable site structure for AI crawlers
- **Improved structured data** - Better JSON-LD schemas throughout

### Other Improvements

- **Custom 404 page** - Helpful error page with popular content links
- **Last Updated dates** - See when each article was last modified
- **Reading time estimates** - Know how long each tutorial takes
- **Google Search Console** - Now verified for better search insights

## Technical Details

These changes include:

- Static OG images for better LinkedIn compatibility
- Breadcrumb schemas for navigation
- Course schemas for learning tracks
- Article schemas for content pages

## What's Next

We're continuing to improve the platform based on user feedback. Stay tuned for more learning content and features!
