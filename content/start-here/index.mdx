## What Makes Claude Code Different?

Most AI coding tools work like a chatbotâ€”you copy code in, get an answer, paste it back. That's tedious and error-prone.

Claude Code works differently:

<Diagram title="Claude Code vs Traditional AI" type="flowchart">
{`flowchart TB
    subgraph traditional["Traditional AI Chatbots"]
        A1[Copy code] --> A2[Paste into chat]
        A2 --> A3[Get response]
        A3 --> A4[Copy back]
        A4 --> A5[Hope it works]
    end

    subgraph claude["Claude Code"]
        B1[Ask Claude] --> B2[Claude reads your files]
        B2 --> B3[Claude writes changes directly]
        B3 --> B4[You review & approve]
    end
`}
</Diagram>

**With Claude Code, you can:**

- Ask questions about code you don't understand
- Get changes applied directly to your files
- Run terminal commands without memorizing syntax
- Debug errors by just pasting them
- Generate entire features with context of your whole project

<Callout type="note">
Claude Code runs locally in VS Code but connects to Anthropic's API. Your code stays on your machineâ€”only what you send to Claude goes to the cloud.
</Callout>
