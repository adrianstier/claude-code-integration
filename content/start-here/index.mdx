# Start Here

Welcome! This guide will get you from zero to fully set up with Claude Code, VS Code, and Git/GitHub.

## What You'll Set Up

By the end of this track, you'll have:

-  VS Code installed and configured
-  Claude Code extension working
-  Git configured with your identity
-  GitHub account connected via SSH
-  Your first repo cloned with Claude templates

## Time Required

- **Mac**: ~60 minutes
- **Windows**: ~75 minutes
