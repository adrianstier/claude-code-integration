This guide will walk you through setting up your Mac for development with Claude Code, VS Code, and Git/GitHub.

Make sure you have administrator access to your Mac and a stable internet connection. This guide will take approximately 60 minutes to complete.

## Step 1: Install VS Code

<Steps>
<Step title="Download VS Code">

1. Visit [https://code.visualstudio.com](https://code.visualstudio.com)
2. Click "Download for Mac"
3. Open the downloaded `.zip` file
4. Drag `Visual Studio Code.app` to your Applications folder

</Step>
<Step title="Open VS Code from Terminal">

To open VS Code from the command line:

1. Open VS Code
2. Press <Kbd>Cmd</Kbd> + <Kbd>Shift</Kbd> + <Kbd>P</Kbd> to open the Command Palette
3. Type "shell command"
4. Select "Shell Command: Install 'code' command in PATH"

**Test it:**

```bash
code --version
```

You should see the VS Code version number.

</Step>
</Steps>

## Step 2: Install Homebrew (Package Manager)

Homebrew makes it easy to install development tools on Mac.

<Steps>
<Step title="Install Homebrew">

Open Terminal (Applications → Utilities → Terminal) and run:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Follow the prompts. You may need to enter your password.

</Step>
<Step title="Add Homebrew to PATH">

After installation, run the commands Homebrew suggests (they'll look like this):

```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
```

**Test it:**

```bash
brew --version
```

</Step>
</Steps>

## Step 3: Install Git

Git is the version control system you'll use to track changes and collaborate.

<Steps>
<Step title="Install Git via Homebrew">

```bash
brew install git
```

</Step>
<Step title="Configure Git">

Set your name and email (use the email you'll use for GitHub):

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

Set VS Code as your default editor:

```bash
git config --global core.editor "code --wait"
```

Set default branch name to `main`:

```bash
git config --global init.defaultBranch main
```

</Step>
<Step title="Create Useful Aliases">

```bash
git config --global alias.st status
git config --global alias.co checkout
git config --global alias.br branch
git config --global alias.cm commit
```

**Test it:**

```bash
git --version
git config --list
```

</Step>
</Steps>

## Step 4: Set Up GitHub

<Steps>
<Step title="Create GitHub Account">

1. Go to [https://github.com](https://github.com)
2. Click "Sign up"
3. Follow the prompts to create your account

</Step>
<Step title="Generate SSH Key">

SSH keys let you connect to GitHub securely without entering your password every time.

**Generate a new SSH key:**

```bash
ssh-keygen -t ed25519 -C "your.email@example.com"
```

When prompted:
- Press <Kbd>Enter</Kbd> to accept the default file location
- Enter a passphrase (optional but recommended)
- Press <Kbd>Enter</Kbd> again to confirm

**Start the SSH agent:**

```bash
eval "$(ssh-agent -s)"
```

**Add your key to the agent:**

```bash
ssh-add ~/.ssh/id_ed25519
```

</Step>
<Step title="Add SSH Key to GitHub">

**Copy your public key:**

```bash
pbcopy < ~/.ssh/id_ed25519.pub
```

This copies your key to the clipboard.

**Add to GitHub:**

1. Go to [https://github.com/settings/keys](https://github.com/settings/keys)
2. Click "New SSH key"
3. Title: "My Mac" (or any descriptive name)
4. Key type: Authentication Key
5. Paste your key (<Kbd>Cmd</Kbd> + <Kbd>V</Kbd>)
6. Click "Add SSH key"

**Test your connection:**

```bash
ssh -T git@github.com
```

<Callout type="success">
You should see: "Hi [username]! You've successfully authenticated..."
</Callout>

</Step>
</Steps>

## Step 7: Install Python (For Data Analysis)

If you plan to do data analysis with Python:

<Steps>
<Step title="Install Python via Homebrew">

```bash
brew install python@3.11
```

</Step>
<Step title="Create a Virtual Environment">

For your first project:

```bash
mkdir ~/my-first-project
cd ~/my-first-project
python3 -m venv venv
source venv/bin/activate
```

You'll see `(venv)` in your prompt when activated.

</Step>
<Step title="Install Common Data Science Packages">

```bash
pip install pandas numpy matplotlib jupyter
```

**Deactivate when done:**

```bash
deactivate
```

</Step>
</Steps>

## Troubleshooting

<Tabs defaultTab="cmd-not-found">
  <TabList>
    <Tab value="cmd-not-found">Command Not Found</Tab>
    <Tab value="permission">Permission Denied</Tab>
    <Tab value="git-push">Git Push Fails</Tab>
    <Tab value="vscode-git">VS Code Can't Find Git</Tab>
  </TabList>
  <TabPanel value="cmd-not-found">

### Command Not Found

If you get "command not found" errors:

1. Close and reopen Terminal
2. Check your PATH:

```bash
echo $PATH
```

3. Ensure Homebrew is in your PATH (Step 2.2)

  </TabPanel>
  <TabPanel value="permission">

### Permission Denied

If you get permission errors with npm:

```bash
mkdir ~/.npm-global
npm config set prefix '~/.npm-global'
echo 'export PATH=~/.npm-global/bin:$PATH' >> ~/.zprofile
source ~/.zprofile
```

  </TabPanel>
  <TabPanel value="git-push">

### Git Push Fails

If `git push` asks for username/password, your SSH key isn't set up correctly. Repeat Step 4.

<Callout type="warning">
Make sure you're using the SSH URL (git@github.com:...) not the HTTPS URL.
</Callout>

  </TabPanel>
  <TabPanel value="vscode-git">

### VS Code Can't Find Git

Open VS Code settings (<Kbd>Cmd</Kbd> + <Kbd>,</Kbd>) and search for "git.path". Set it to:

```
/opt/homebrew/bin/git
```

  </TabPanel>
</Tabs>

## Tips for Mac Users

- Use <Kbd>Cmd</Kbd> + <Kbd>Shift</Kbd> + <Kbd>P</Kbd> frequently in VS Code for the Command Palette
- Terminal shortcuts:
  - <Kbd>Cmd</Kbd> + <Kbd>T</Kbd>: New tab
  - <Kbd>Cmd</Kbd> + <Kbd>K</Kbd>: Clear screen
  - <Kbd>Ctrl</Kbd> + <Kbd>C</Kbd>: Cancel current command
- Keep your system updated: Check for macOS updates regularly
- Use Time Machine for backups before major changes
